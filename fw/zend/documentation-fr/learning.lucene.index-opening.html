Ouverture et création d'index

Toutes les opérations sur l'index (ex: créer un nouvel index, ajouter un document à l'index, supprimer un document, chercher dans l'index) nécessite un objet index. Il peut être obtenu avec l'une des méthodes suivantes.

Example #1 Création d'index Lucene

  1. $index = Zend_Search_Lucene::create($indexPath);

Example #2 Ouverture d'index Lucene

  1. $index = Zend_Search_Lucene::open($indexPath);
