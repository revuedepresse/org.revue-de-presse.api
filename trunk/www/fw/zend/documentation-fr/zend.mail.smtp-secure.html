Sécuriser les transports SMTP

Zend_Mail supporte aussi l'utilisation des connexions SMTP sécurisées via TLSS ou SSL. Ceci peut être activé en passant le paramètre "ssl" ou "tls" au tableau de configuration du constructeur de Zend_Mail_Transport_Smtp. Un port peut optionnellement être fourni, sinon il vaut par défaut 25 pour TLS et 465 pour SSL.

Example #1 Activer une connexion sécurisée dans Zend_Mail_Transport_Smtp

  1. $config = array('ssl' => 'tls',
  2.                 'port' => 25); // Port optionel fourni
  3.  
  4. $transport = new Zend_Mail_Transport_Smtp('mail.server.com', $config);
  5.  
  6. $mail = new Zend_Mail();
  7. $mail->setBodyText('Ceci est le texte de l\'email.');
  8. $mail->setFrom('emetteur@test.com', 'Un émetteur');
  9. $mail->addTo('destinataire@test.com', 'Un destinataire');
  10. $mail->setSubject('Sujet de test');
  11. $mail->send($transport);
